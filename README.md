# Streaming Video Recorder
Use OpenCV-Python and Flask to create a web streaming video recorder for camera.

## Environment
* Python  >=3.6 
* OpenCV 3.3.0

## How to Run 
1. Install **Flask**:

    ```
    pip install flask
    ```

2. Run the app:

    ```
    python server.py
    ```


## Reference
* https://github.com/log0/video_streaming_with_flask_example

## Dockerizing
```
docker-compose up --build
```
